# agent-matter-shinning-3c588

## Общее описание
TravelWeather Assistant — интеллектуальный агент, который помогает туристам планировать маршруты с учётом погодных условий. Агент предоставляет:
- прогноз погоды на текущий день и неделю вперед;
- рекомендации по одежде;
- уведомления об изменениях прогноза;
- советы по выбору транспорта в зависимости от погоды;
- прогноз погоды в другом регионе, если турист туда направляется.

## Архитектура агента
### Компоненты:
- **Orchestrator** — управляет логикой агента.
- **LLM Decision Engine** — анализирует запросы и принимает решения.
- **Weather API Tool** — получает текущую и недельную погоду.
- **Route Analysis Tool** — анализ маршрутов и транспорта.
- **Transport Recommendation Engine** — рекомендации по транспорту.
- **Alerts Manager** — уведомления об изменении погоды.
- **Memory Layer** — хранение контекста и настроек пользователя.

## Workflow
1. Получение пользовательского запроса.
2. Определение намерения (маршрут, прогноз, регион, одежда, транспорт).
3. Запрос к Weather API.
4. Анализ погодных условий.
5. Формирование рекомендаций.
6. Сохранение прогноза для отслеживания изменений.
7. Уведомления при изменении данных.

## Инструменты (Tools)
### getCurrentWeather
Получение текущей погоды по локации.

### getWeeklyForecast
Прогноз погоды на 7 дней.

### analyzeRoute
Определение расстояния и подходящего транспорта.

### notifyWeatherChange
Уведомление пользователя об изменениях погоды.

## Память
- **Short-term**: текущий маршрут, регион, прогноз (TTL 30 мин).
- **Long-term**: предпочтения пользователя по одежде, транспорту и частым маршрутам.

## Модель
- LLM: gpt-4.1
- temperature: 0.2
- style: дружелюбный, информативный

## Безопасность
- Нет хранения лишних персональных данных.
- Использование авторитетных источников погоды.
- Предупреждения о погодных рисках.
